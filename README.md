# practicesoftwaretesting-javaclient
Java clients for the PracticeSoftwareTesting web application

There are 5 versions of the application in different states of completion:
* Sprint 1: Product listing  
  Features: product overview page, product category page, product detail page, contact form  
  Website: https://v1.practicesoftwaretesting.com  
  API documentation: https://api-v1.practicesoftwaretesting.com/api/documentation

* Sprint 2: Paginate, filter & search  
  Website: https://v2.practicesoftwaretesting.com  
  API documentation: https://api-v2.practicesoftwaretesting.com/api/documentation

* Sprint 3: Order placement  
  Features: cart, checkout flow  
  Website: https://v3.practicesoftwaretesting.com  
  API documentation: https://api-v3.practicesoftwaretesting.com/api/documentation

* Sprint 4: Manage account  
  Features: manage customer account  
  Website: https://v4.practicesoftwaretesting.com  
  API documentation: https://api-v4.practicesoftwaretesting.com/api/documentation

* Sprint 5: Admin account  
  Features: PIM, reporting, managing all entities  
  Website: https://practicesoftwaretesting.com  
  API documentation: https://api.practicesoftwaretesting.com/api/documentation  

In this repo these are reflected in the branches for each sprint.
